# BugShield Pest Control

Professional pest control company landing page for the greater Seattle & Everett, WA area.

## Live Demo

[bugshield-pest-control.vercel.app](https://bugshield-pest-control.vercel.app)

## Features

- Professional dark green + amber color scheme
- Fully responsive, mobile-first design
- Sticky navigation with smooth scrolling
- 6 detailed pest control service cards
- Trust indicators (Licensed & Insured, Eco-Friendly, Same-Day Service)
- 3-step process section (Inspection, Treatment, Protection)
- Customer testimonials with star ratings
- Service areas listing (Seattle/Everett metro)
- Call-to-action sections with contact info
- Complete SEO metadata with Open Graph tags

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Language:** TypeScript
- **Deploy:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deployed automatically via Vercel on push to `main`.
